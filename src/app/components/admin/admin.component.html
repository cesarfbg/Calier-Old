Ajustemos las cosas

Bienvenido/a al panel de administración. En esta sección podrás realizar los cambios y ajustes en el contenido que presenta la pagina web.


En la sección de Productos se encuentra el área de gestión de productos, donde se agrega, edita o elimina cualquiera de ellos.


Seguido se encuentra Blog, donde se gestiona las diferentes entradas del blog de Calier. Cada entrada tiene sus diferentes campos incluyendo uno llamado Status que determina si la entrada se muestra o no al público general.


Por último la sección Contenido que será el area de administración de contenido que se muestra en cada pestaña del sitio web Calier. Para cada sub-sección se encuentra los diferentes campos para una perzonalización mas detallada permitiendo mejoras cuando se requiera.
